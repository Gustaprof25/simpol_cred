Simpol Cred — Landing Page

Como executar localmente:

1. Abra `index.html` no navegador (duplo clique) ou rode um servidor estático.

Ex.: com Python 3:

```bash
python -m http.server 8000
# então acesse http://localhost:8000
```

O projeto usa Bootstrap e Font Awesome via CDN. O simulador é um script simples em `assets/js/main.js`.
